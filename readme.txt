       





      PROIECT MEMORIE CULORI  - ARDUINO + PROCESSING




Autori :
 
Pavel Alexandru (30227)
Reclaru Dragos (30226)






FUNCTIONALITATE:

- Joc interactiv pentru 2 playeri

- Scor, vieti si nivel afisate in timp real in interfata grafica

- Realizat cu ajutorul mediului Arduino si Processing

- Meniu interactiv PC, usor de folosit si lizibil

- Aplicatie pe Android destinata coenctivitatii Bluetooth pentru un al 2-lea jucator

- Implementarea hardware este bazata pe Microchipul AVR (8 b), gasit pe placa Arduino Uno 

- Sunete emise pentru fiecare actiune speciala din joc, lucru realizat cu ajutorul unui Buzzer 

- Feedback instant la apasarea butoanelor, timeout-ul conexiunii Bluetooth fiind de 50ms, lucru care faciliteaza viteza

- Depistarea apasarii unei culori de mai multe ori si normalizarea acesteia la o singura instanta
